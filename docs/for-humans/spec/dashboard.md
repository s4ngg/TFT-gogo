# Dashboard — 메인

## 개요

홈 화면. 소환사 검색 진입점, 메타 스냅샷, 사이드바 위젯(파티 모집·채팅·AI 덱 추천)으로 구성된다.
검색 없이도 메타 스냅샷이 기본으로 노출된다.

> 검색 후 이동하는 소환사 전적 상세는 [`summoner.md`](summoner.md) 참조.

---

## 레이아웃

| 영역 | 컴포넌트 | 비고 |
|------|---------|------|
| 상단 좌측 | PatchMetaCard | 최신 패치 메타 + 패치노트 이동 버튼 |
| 상단 우측 | SummonerSearchCard | 소환사명 검색 입력창 |
| 중단 전체 | MetaSnapshot | 덱 메타 스냅샷 테이블 |
| 우측 사이드바 | PartyFinderCard | 파티 모집 미리보기 |
| 우측 사이드바 | LiveChat | 라이브 채팅 미리보기 |
| 우측 사이드바 | AiDeckRecommend | AI 덱 추천 미리보기 |

## 소환사 검색 (SummonerSearchCard)

- `소환사명#태그` 형식 입력 후 검색 시 `/summoner/{name}/{tag}` 페이지로 이동
- `#` 미입력 시 태그 기본값 `KR1` 적용
- **인기 검색 태그**: 고정 목록 4개 버튼 클릭으로 바로 이동 (최근 검색어 저장 기능 없음)

## 메타 스냅샷 (MetaSnapshot)

### 필터 탭

| 탭 | 조건 |
|----|------|
| 종합 | 전체 덱 |
| 상위권 | grade = S 또는 A+ |
| 마스터+ | TOP4 ≥ 68% |

### 정렬 옵션

| 옵션 | 정렬 방향 |
|------|---------|
| 승률 | 내림차순 |
| TOP 4 | 내림차순 |
| 평균 등수 | 오름차순 |

### 덱 테이블 표시 항목

| 항목 | 내용 |
|------|------|
| 순위 | `rank` |
| 티어 등급 | S / A+ / A / B / C / D (`grade`) |
| 덱 이름 | 주력 시너지·유닛 조합 자동 생성 |
| 주요 시너지 | `TraitHexBadge` 컴포넌트 |
| 주요 유닛 | `ChampionCard` 컴포넌트 |
| 승률 | `winRate` |
| TOP 4 | `top4` |
| 평균 등수 | `avgPlace` |

- 덱 클릭 시 덱 상세 페이지로 이동 (`ChevronRight` 버튼)
- 동점 시 `rank` 기준 오름차순 2차 정렬

## 미결 사항

| 항목 | 우선순위 |
|------|---------|
| 인기 검색 태그 → 실제 검색 빈도 기반 동적 목록 전환 | 낮음 |
| 메타 스냅샷 데이터 백엔드 API 연동 (현재 목 데이터) | 높음 |
| 사이드바 위젯 실시간 데이터 연동 | 중간 |
