# Party — 파티원 찾기 & 실시간 채팅

## 개요

파티원 모집 게시판과 실시간 채팅을 한 화면에서 제공하는 커뮤니티 기능.
좌측 파티원 찾기 패널과 우측 채팅 패널이 나란히 배치된다.

---

## 파티원 찾기

### 필터 탭

`partyFilters.ts`에 정의된 4가지 탭으로 모집글을 필터링한다.

| 탭 | 설명 |
|----|------|
| 전체 | 모든 모드 표시 (기본값) |
| 랭크 | 랭크 게임 모집글만 |
| 일반 | 일반 게임 모집글만 |
| 커스텀 | 커스텀 게임 모집글만 |

- 탭 클릭 즉시 목록 재필터
- 모집글 작성 후 등록 시 탭이 `전체`로 초기화됨

### 검색

- 검색 대상 필드: 제목, 모드, 티어, 설명, 태그
- 타이핑 즉시 반영 (form submit 불필요)
- 필터 탭 + 검색 조건 AND 조합 적용
- 결과 없을 때 빈 상태 안내: `조건에 맞는 모집글이 없습니다.`

### 모집글 카드

| 항목 | 내용 |
|------|------|
| 아이콘·색조 | 모드·티어 기반 자동 결정 (아래 표 참조) |
| 제목 | 모집글 제목 |
| 설명 | 플레이 스타일 / 요청사항 |
| 모드 배지 | 랭크 / 일반 / 커스텀 |
| 티어 조건 | 마스터+ / 다이아+ / 플래티넘+ / 제한 없음 |
| 인원 | `current/total` 형식 |
| 마감 시간 | 등록 시각 기준 상대 표시 (`방금 등록` 등) |
| 태그 | 쉼표 구분 입력, 최대 4개 |
| 상태 | 모집중 / 대기중 / 마감 |

**아이콘·색조 자동 결정 규칙 (`getPostStyle`)**

| 조건 | 아이콘 | 색조 |
|------|--------|------|
| 모드 = 일반 | Sparkles | cyan |
| 모드 = 커스텀 | Swords | gold |
| 모드 = 랭크 + 티어에 "다이아" 포함 | Leaf | green |
| 모드 = 랭크 (나머지) | Crown | purple |

### 모집 상태 관리

- `status` 필드: `모집중` → 인원 마감 시 `대기중`으로 전환
- 카드 헤더는 `isFull` 여부를 실시간 계산해 `모집중` / `대기중` / `마감` 표시
- **참여 버튼 상태**

| 상태 | 버튼 텍스트 | disabled |
|------|------------|---------|
| 참여 가능 | 참여 | false |
| 이미 이 파티 참여 중 | 참여중 | false (취소 가능) |
| 정원 마감 | 마감 | true |
| 다른 파티 참여 중 | 잠김 | true |

- 한 번에 한 파티만 참여 가능 (`joinedPostId` 단일 상태)
- 참여 / 취소 시 인원 수 즉시 반영 (`capacity` 문자열 갱신)
- 참여 시 채팅 "파티 모집" 채널에 자동 메시지 전송 + 채널 포커스 이동

### 모집글 작성 폼

| 필드 | 유형 | 비고 |
|------|------|------|
| 제목 | text input | 필수 |
| 모드 | select | 랭크 / 일반 / 커스텀 |
| 티어 조건 | select | 마스터+ / 다이아+ / 플래티넘+ / 제한 없음 |
| 모집 인원 | text input | `N` 또는 `current/total` 형식 허용 (`normalizeCapacity` 정규화) |
| 태그 | text input | 쉼표 구분, 최대 4개 슬라이스 |
| 설명 | textarea | 필수 |

- 제목 또는 설명 미입력 시 에러 메시지 표시
- 등록 성공 시 목록 상단 삽입, 폼 초기화, 필터 탭 `전체`로 리셋

## 실시간 채팅

### 채널 목록

| 채널 | 기본 참여 인원 |
|------|--------------|
| 일반 | 1,234 |
| 덱 공략 | 856 |
| 파티 모집 | 622 |
| 질문 & 답변 | 741 |

- 채널별 참여 인원 수, 마지막 메시지 미리보기 표시
- 채널 클릭으로 활성 채널 전환
- 온라인 배지: 페이지 헤더에 총 온라인 인원 표시 (현재 4,113 고정)

### 채팅 창

- 활성 채널 메시지만 필터링해 표시
- 메시지 항목: 닉네임, 티어, 시각, 내용
- 내 메시지(`isMine: true`)는 별도 스타일로 구분
- 메시지 없는 채널: `아직 이 채널에는 메시지가 없습니다.` 안내
- 전송 폼: 빈 문자열 전송 차단, 전송 후 입력창 초기화
- 전송 시 채널 `lastMessage` 즉시 갱신
