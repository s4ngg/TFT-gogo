# Decks — 덱 목록 & 상세

## 개요

현재 패치의 추천 덱 목록과 각 덱의 상세 운영법을 제공한다.

---

## 덱 목록 (`/decks`)

- 현재 패치 추천 덱 목록을 확인할 수 있다
- 티어별 그룹핑으로 표시된다 (S / A / B / C / D)
- 랭크 구간 필터를 선택할 수 있다 (마스터+ / 다이아+ / 에메랄드+)
- 덱 이름: `[주요 시너지] [캐리 챔피언명]` 형식으로 자동 생성되며, 관리자가 직접 수정할 수 있다
- 각 덱에 챔피언 아이콘, 시너지 뱃지, 승률/TOP4/평균 등수/픽률을 표시한다
- 챔피언 목록은 해당 덱의 최종 운영 레벨 기준으로만 표시한다
  - 5코스트 캐리 덱 → Lv.9 기준 (5코스트 포함)
  - 4코스트 캐리 덱 → Lv.8 기준 (1~4코스트)
  - 3코스트 리롤 덱 → Lv.7 기준 (1~4코스트)
  - 1~2코스트 리롤 덱 → Lv.8 기준 (1~4코스트)

---

## 덱 상세 (`/decks/:deckId`)

### 추천 배치 (헥스 보드)

- Lv.5~9 탭으로 레벨별 유닛 구성 확인
- 레벨 탭 변경 시 배치판과 시너지 구성이 동시에 업데이트된다
- 유닛은 코스트 오름차순으로 정렬 (빌드업 순서 반영)
- 관리자가 배치판 포지션을 직접 지정한 경우, 자동 배치 대신 관리자 배치를 우선 표시한다

### 시너지 구성

- 현재 선택 레벨의 visibleUnits 기반으로 계산 (CDragon 트레이트 데이터 활용)
- 레벨 탭에 따라 동적으로 변경

### 추천 아이템

- 캐리 유닛별 핵심 아이템 3개 표시

### 증강 추천

- Riot API에서 증강 데이터를 제공하지 않음 (TFT-MATCH-V1 미지원 확인)
- **향후 관리자 수동 큐레이션으로 제공 예정**
