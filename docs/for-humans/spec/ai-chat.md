# AI 채팅 — 소환사 전적 기반 AI 어시스턴트

## 개요

소환사 상세 페이지에서 TFT 관련 질문에 AI가 답변하는 채팅 기능.
전적 데이터를 컨텍스트로 주입하여 개인화된 분석을 제공한다.

---

## 기능 목록

- 소환사 전적 데이터 기반 개인화 질문 응답 (챔피언 통계, 시너지 분석, 순위 요약 등)
- 일반 TFT 지식 질문 응답 (덱 운영법, 아이템 조합, 메타 조언 등)
- 데이터 한계 시 솔직한 안내 (아이템/증강체/LP 등 미지원 항목)

---

## 기술 구조

- 프론트엔드 → Spring 백엔드(프록시) → AI 서버(FastAPI) → OpenAI GPT-4o-mini
- 컨텍스트: 초기 20게임 기준 통계 (챔피언/시너지 사전 집계, 개별 매치 기록)
- 챔피언 이름: CDragon locale로 한국어 변환

---

## 답변 가능 범위

### 답변 가능
- 전적 요약 (평균 등수, TOP4율, 1등률)
- 개별 게임 순위 및 사용 덱
- 많이 사용한 챔피언/시너지 통계
- 일반 TFT 지식 (덱 추천, 운영 팁 등)

### 답변 불가 (데이터 미포함)
- 아이템 빌드 정보
- 증강체 선택 정보
- LP 변동 및 승급/강등 이력
- 다른 소환사 전적
- 전체 서버 메타 통계
