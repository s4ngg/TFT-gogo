# docs/for-humans/

팀원이 읽는 문서 폴더.

| 파일/폴더 | 내용 |
|------|------|
| `constitution.md` | 팀 개발 프로세스, 컨벤션 원문 |
| `spec.md` | 서비스 전체 기능 한눈에 보기 (인덱스) |
| `spec/` | 기능별 상세 스펙 |
| `plan.md` | 개발 계획, 마일스톤 |
| `tasks.md` | 작업 목록, 진행 상황 |
| `checklist.md` | 배포, 작업 체크리스트 |
| `screenshots/` | UI 스크린샷 |
| `frontend/` | 프론트엔드 전용 문서 |
| `backend/` | 백엔드 전용 문서 |

> **AI 도구는 이 폴더를 열지 않는다.** `docs/for-ai/README.md`를 참고.
