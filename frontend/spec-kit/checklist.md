# checklist.md — 검증 정의

> 산출물이 올바른가. 각 기능의 완료 조건을 정의한다.
> PR 머지 전 해당 기능의 체크리스트를 모두 통과해야 한다.

---

## 공통 레이아웃 / 라우팅

- [ ] `/` 접속 시 MatchSearch 페이지가 기본으로 렌더링된다
- [ ] 헤더, 사이드바가 모든 페이지에서 동일하게 노출된다
- [ ] 사이드바에서 메뉴 클릭 시 해당 페이지로 이동한다
- [ ] 현재 페이지에 해당하는 메뉴가 사이드바에서 하이라이트된다
- [ ] 존재하지 않는 경로 접근 시 404 페이지가 노출된다
- [ ] 라우팅 세팅 PR 파일 수 20개 이하 (또는 예외 사유 PR 설명 기재)

---

## Dashboard — 메인 (소환사 전적 검색)

- [ ] 소환사명#태그 입력 후 엔터/버튼 클릭 시 검색이 실행된다
- [ ] 검색 중 로딩 상태가 표시된다
- [ ] 검색 결과로 소환사 카드(티어, LP, 승률)가 노출된다
- [ ] 검색 결과가 없을 때 빈 상태 메시지가 노출된다
- [ ] API 오류 발생 시 에러 메시지가 노출된다
- [ ] 메타 스냅샷이 페이지 진입 시 기본으로 노출된다
- [ ] 종합 / 상위권 / 마스터+ 탭 클릭 시 목록이 전환된다
- [ ] 최근 검색어 태그 클릭 시 해당 소환사로 검색된다

- [ ] 덱 클릭 시 상세 페이지로 이동한다

- [ ] 추천 결과로 유닛 구성, 아이템 배치가 노출된다
- [ ] 분석 실패 시 에러 메시지가 노출된다

## Guide — 게임 가이드

- [ ] 패치 변경사항 목록이 정상 노출된다
- [ ] 유닛/시너지/아이템 정보가 조회된다

---

## Community — 커뮤니티

### 파티원 찾기

- [ ] 모집글 목록이 전체 / 랭크 / 일반 / 커스텀 탭으로 필터링된다
- [ ] 티어·모드·키워드 검색 시 결과가 필터링된다
- [ ] 글 작성 폼에서 필수 항목 미입력 시 유효성 오류가 표시된다
- [ ] 글 작성 완료 시 목록에 즉시 반영된다
- [ ] 각 모집글에 모집 상태, 인원, 마감 시간, 태그가 표시된다
- [ ] 참여 버튼 클릭 시 정상 동작한다

### 실시간 채팅

- [ ] 채널 목록(#일반, #덱 공략, #파티 모집, #질문 & 답변)이 노출된다
- [ ] 채널별 접속 인원 수가 표시된다
- [ ] 채널 클릭 시 해당 채널 메시지가 노출된다
- [ ] 실시간으로 메시지를 주고받을 수 있다
- [ ] 서버 전체 온라인 유저 수가 표시된다

---

## PatchNotes — 패치 노트

- [ ] 최신 패치 노트가 정상 노출된다
- [ ] 챔피언/아이템/시너지 변경사항이 노출된다
- [ ] 이전 패치 히스토리 목록이 조회된다
