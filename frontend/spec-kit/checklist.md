## SummonerDetail — 소환사 전적 상세

- [ ] 소환사 카드(소환사명, 태그, 티어, LP, 승수, 패수, 승률, 평균 순위, TOP4율)가 정상 노출된다
- [ ] 존재하지 않는 소환사 검색 시 빈 상태(Empty State) 안내가 노출된다
- [ ] 전적 업데이트 버튼 클릭 시 최신 데이터로 갱신된다
- [ ] 갱신 중 로딩 상태가 표시된다
- [ ] 갱신 완료 후 전적 목록이 자동 리페치된다
- [ ] 갱신 실패 시 에러 메시지가 노출된다
- [ ] 갱신 중 버튼이 disabled 처리된다
- [ ] 많이 플레이한 시너지 / 챔피언 통계 섹션이 노출된다
- [ ] 최근 30게임 요약이 정상 노출된다
  - [ ] 순방확률 도넛 차트(퍼센트)가 표시된다
  - [ ] 순방확률 W/L 통계와 평균 순위가 표시된다
- [ ] 최근 전적 목록이 정상 노출된다
  - [ ] 순위(1위~8위 형식), 덱 이름이 표시된다
  - [ ] 각 전적 행에 게임 유형 배지(랭크 / 일반)가 표시된다
  - [ ] 전적 목록을 게임 유형(전체 / 랭크 / 일반)으로 필터링할 수 있다
  - [ ] 게임 시간(방금 전 / 몇 분 전 / 몇 시간 전 등)이 표시된다
  - [ ] LP 변동값은 표시하지 않는다 (Riot API 미제공)
  - [ ] 유닛 이미지 및 유닛별 아이템 이미지(최대 3개)가 표시된다
  - [ ] 챔피언·아이템 이미지 호버 시 이름 툴팁이 표시된다
- [ ] 전적 행 클릭 시 8명 전원 상세 패널이 펼쳐진다
  - [ ] 순위, 소환사명, 스테이지(last_round 기반 Spring 변환값), 시너지, 챔피언(아이템 포함)이 표시된다
  - [ ] 순위 색상이 적용된다 (1위 금색, 2위 은색, 3위 동색, 4위 이하 기본색)
  - [ ] 시너지 아이콘이 소환사 카드 "많이 플레이한 시너지"와 동일한 CDN 이미지로 표시된다
  - [ ] 증강은 표시되지 않는다 (Riot API 미제공 확정)
  - [ ] 킬수, 잔여골드가 표시된다
  - [ ] 내 행이 하이라이트된다
- [ ] "30개 더 보기" 버튼으로 추가 전적을 불러올 수 있다
- [ ] 로그인 없이 조회 가능하다

## Guide — 게임 가이드

- [ ] 시너지 / 아이템 / 증강체 / 챔피언 탭이 정상 노출된다
- [ ] 각 탭에서 키워드 검색 시 목록이 필터링된다
- [ ] 검색어 또는 필터 변경 시 목록이 1페이지로 초기화된다
- [ ] 탭별 페이지네이션 표시 개수가 스펙과 일치한다
  - [ ] 시너지: 페이지당 6개
  - [ ] 아이템: 페이지당 5개
  - [ ] 증강체: 페이지당 5개
  - [ ] 챔피언: 페이지당 10개
- [ ] 페이지 번호는 한 번에 최대 5개까지만 노출된다
- [ ] 시너지 가이드에서 이름, 유형, 활성 단계, 효과, 필요 챔피언, 운영 팁이 노출된다
- [ ] 아이템 통계에서 승률, TOP4, 픽률, 평균 등수, 추천 챔피언, 조합 메모가 노출된다
- [ ] 아이템 목록이 승률, TOP4, 픽률, 평균 등수 기준으로 정렬된다
- [ ] 증강체 가이드에서 티어, 승률, 픽률, 평균 등수, 설명, 보상표, 운영 플랜이 노출된다
- [ ] 챔피언 가이드에서 코스트 필터, 시너지, 역할, 포지션, 3신기가 노출된다
- [ ] 챔피언 카드 hover 시 핵심 스탯이 노출된다
- [ ] 챔피언 상세 모달에서 스탯, 시너지, 추천 아이템이 노출된다
- [ ] 챔피언 즐겨찾기와 최근 확인한 가이드로 빠르게 이동할 수 있다
- [ ] 즐겨찾기 보기 상태에서 개별 해제와 전체 목록 복귀가 가능하다
- [ ] 시너지 / 아이템 추천 챔피언 / 챔피언 카드 클릭 시 관련 가이드로 이동한다
- [ ] 조건에 맞는 결과가 없을 때 빈 상태가 노출된다
- [ ] API 연동 시 로딩 / 에러 / 재시도 상태가 노출된다
- [ ] 접근성 속성이 적용된다
  - [ ] 탭 `aria-selected`
  - [ ] 페이지네이션 `aria-current`
  - [ ] 즐겨찾기 버튼 `aria-pressed`
  - [ ] 챔피언 상세 모달 focus 이동/복귀

## PatchNotes — 패치 노트

- [ ] 최신 패치 요약 정보가 정상 노출된다
  - [ ] 패치 버전
  - [ ] 적용일
  - [ ] 패치 설명
  - [ ] 전체 변경 건수
  - [ ] 핵심 영향 변경 건수
  - [ ] 상향 / 하향 항목 수
- [ ] 선택된 패치 대표 이미지가 노출된다
- [ ] 대표 이미지 로딩 실패 시 fallback 이미지가 노출된다
- [ ] 챔피언 / 시너지 / 아이템 / 증강체 / 시스템 변경사항이 카테고리별로 노출된다
- [ ] 변경사항 카드에 카테고리, 변경 유형, 영향도, 대상 이름, 요약, 태그, 이미지가 노출된다
- [ ] 상세 보기 클릭 시 변경 전 / 변경 후 비교 영역이 펼쳐진다
- [ ] 검색어, 카테고리, 변경 유형, 영향 높음 필터가 정상 동작한다
- [ ] 검색어 또는 필터 변경 시 목록이 1페이지로 초기화된다
- [ ] 변경사항 목록은 페이지당 5개씩 노출된다
- [ ] 페이지 번호는 한 번에 최대 5개까지만 노출된다
- [ ] 조건에 맞는 변경사항이 없을 때 빈 상태가 노출된다
- [ ] 이전 패치 히스토리 목록이 노출된다
- [ ] 히스토리 항목 클릭 시 선택 패치 기준으로 요약, 목록, 카운트가 갱신된다
- [ ] 선택된 히스토리 항목은 active 상태로 표시된다
- [ ] 패치 버전 변경 시 페이지와 펼쳐진 상세 비교 상태가 초기화된다
- [ ] API 연동 시 히스토리 / 변경사항 로딩 상태가 노출된다
- [ ] API 오류 발생 시 에러 메시지와 재시도 버튼이 노출된다
- [ ] 접근성 속성이 적용된다
  - [ ] 히스토리 선택 버튼 `aria-pressed`
  - [ ] 페이지네이션 `aria-current`
  - [ ] 상세 보기 `aria-expanded`
